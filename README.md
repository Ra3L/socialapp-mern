# socialapp.vercel.app

MERN stack social media app

## Tech stack

- React
- Node.js
- Express.js
- MongoDB
- Vercel
- Material UI
- HTML5

## Run Project Locally

### Prerequisites

- Node
- MongoDB
- npm

## Clone or Download

```bash
$ git clone https://github.com/Ra3L/mern_socialapp.git
$ cd mern_socialapp
```

It requires client and server runs concurrently in separate terminal to make it work.

### Client-side Usage

```bash
$ cd client
$ npm install
$ npm run dev
```

### Server-side Usage

```bash
$ cd server
$ npm install
$ npm run start
```

It requires JWT_SECRET in .env to connect to MongoDB
